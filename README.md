## 特性

- 楼中楼自动折叠
- 依序据总贴贴数/时间正序/时间倒序排
- 允许用户设定**最低有效字数**和**精选评论数**等

> 提示：精选评论区中的留言至少会有一份贴贴，如果最终精选评论区内的回复不足10条（不论设定导致还是贴贴的评论太少）会自动展开普通评论区。
